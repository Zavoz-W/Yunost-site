   → https://fonts.google.com/specimen/Montserrat
   → Скачать: Weight 700 (Bold), формат woff2

2. Roboto-Regular.woff2
   → https://fonts.google.com/specimen/Roboto
   → Скачать: Weight 400 (Regular), формат woff2

3. Roboto-Bold.woff2
   → https://fonts.google.com/specimen/Roboto
   → Скачать: Weight 700 (Bold), формат woff2

4. Inter-Regular.woff2
   → https://fonts.google.com/specimen/Inter
   → Скачать: Weight 400 (Regular), формат woff2

5. Inter-Bold.woff2
   → https://fonts.google.com/specimen/Inter
   → Скачать: Weight 700 (Bold), формат woff2

6. Nunito-Bold.woff2
   → https://fonts.google.com/specimen/Nunito
   → Скачать: Weight 700 (Bold), формат woff2

--------------------------------------------------
КАК СКАЧАТЬ (через fontsource или другой источник):
--------------------------------------------------
Например через сайт https://fontsource.org/:
- Montserrat → https://fontsource.org/fonts/montserrat
- Roboto     → https://fontsource.org/fonts/roboto
- Inter      → https://fontsource.org/fonts/inter
- Nunito     → https://fontsource.org/fonts/nunito

Или через Google Fonts напрямую — зайдите на fonts.google.com,
выберите нужный шрифт → нажмите "Download family" → 
из архива возьмите нужные .ttf файлы и конвертируйте в .woff2
через https://cloudconvert.com/ttf-to-woff2

После добавления файлов в папку fonts/ шрифты подключатся
автоматически через @font-face в style.css (без Google Fonts).
